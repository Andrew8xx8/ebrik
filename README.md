# Ebrik

Simple HTTP web server. By default serves files

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ebrik'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install Ebrik

## Usage

To serve files run:

    $ bin/run

To serve your `config.ru` run:

    $ bundle exec rackup -o 0.0.0.0 -p 1234 -s ebrik
